windowfactory

A global variable exposed on windows to access the windowfactory-related API.
Properties:
Name Type Description
version string version of windowfactory
runtime object contains runtime-specific info
Properties
Name Type Description
name string name of runtime (ie. Chrome, IE, OpenFin, Electron, ect)
version string
isBrowser boolean
isElectron boolean
isOpenFin boolean
isDesktop boolean is this a desktop OS
isMobile boolean is this a mobile OS
isLauncher boolean is window the launcher (true if it is the Browser's launch window)
isMain boolean is window the main (true if window started app, Electron doesn't have one atm)
Window Window
